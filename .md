## Задание 2
Рассматривается множество целых чисел, принадлежащих отрезку [1125; 7856], которые делятся а 3 и не делятся на 7 и 5. Найдите количество таких чисел.

count = 0

## Задание 6
В программу вводится натуральное число N, а затем N натуральных чисел. Требуется найти сумму чисел, которые делятся на 7 и являются трехзначными. В результате программа должна вывести одно число: сумму требуемых чисел.

## Задание 7
В программу вводится число N, после чего - N натуральных чисел, не превышающих 1000. Программа должна найти и вывести на экран максимальное число, кратное 7 и не кратное 9.
```
n = int(input())
max_num = 0  # начальное значение, меньше любого возможного числа

for _ in range(n):
    x = int(input())
    # проверяем условие: кратно 7 и не кратно 9
    if x % 7 == 0 and x % 9 != 0:
        if x > max_num:
            max_num = x

# если ни одно число не подошло, max_num останется равным 0
print(max_num)
```

## Задание 8
В программу вводится число N, после чего - N натуральных чисел, не превышающих 1000. Программа должна найти и вывести на экран минимальное двузначное число, восьмеричная запись которого оканчивается на 3.
```
def main():
    n = int(input())
    min_number = None
    for _ in range(n):
        x = int(input())
        if 10 <= x <= 99 and x % 8 == 3:
            if min_number is None or x < min_number:
                min_number = x
    if min_number is None:
        print(0)
    else:
        print(min_number)

if __name__ == "__main__":
    main()
```
* Пример ввода:
4
11
25
83
19

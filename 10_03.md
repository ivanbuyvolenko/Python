1.  АААА

4.  АААР

5.  ААЕА

...

 

Под каким номером в списке идёт первое слово, в котором нет буквы А?
###
```
a = {0: "А", 1: "Е", 2: "К", 3: "Р"}
k = 0
for i in range(0, len(a)):
    for j in range(0, len(a)):
        for g in range(0, len(a)):
            for m in range(0, len(a)):
                s = a[i] + a[j] + a[g] + a[m]
                k += 1
                if s.count('А') == 0:
                    print(k)  # Возьмём первое число, которое выдаст программа
```

# Задание 8 типа №16385
## Михаил составляет 5-⁠буквенные коды. В кодах разрешается использовать только буквы А, Б, В, Г, Д, при этом код не может начинаться с гласной и не может содержать двух одинаковых букв подряд. Сколько различных кодов может составить Михаил?
###
```
import itertools 
alphabet = "АБВГД"
ar = itertools.product(alphabet, repeat=5) #Размещение с повторением
arl = []
for i in ar:
    arl.append(list(i))
count = 0
for e in arl:
    flag = True
    for i in range(len(e) - 1):
        if e[0] == 'А' or e[i] == e[i + 1]:
            flag = False
    if flag == True: count += 1
print(count)
```
## Ответ: 1024.

# Задание 8 типа №33510
## Тимофей составляет 5-⁠буквенные коды из букв Т, И, М, О, Ф, Е, Й. Буква Т должна входить в код не менее одного раза, а буква Й  — не более одного раза. Сколько различных кодов может составить Тимофей?
###
```
from itertools import product
s = 'ТИМОФЕЙ'
words = []
for w in product(s, repeat=5):
    if ('Т' in w) and w.count('Й') <= 1:
        words.append(w)
print(len(words))
```
## Ответ: 8006.
